------------------------------------------------------------
🎨 COMO PERSONALIZAR AS CORES (IMPORTANTE)
------------------------------------------------------------
O template utiliza variáveis CSS para facilitar a troca de 
identidade visual. Abra o arquivo 'style.css' e altere as 
primeiras linhas no bloco :root:

--primary: #8a2be2;      <-- Mude aqui para sua cor principal
--dark-bg: #0a0a0c;      <-- Cor do fundo (Dark Mode)
--card-bg: #16161a;      <-- Cor dos cards e seções

------------------------------------------------------------
🛠 COMO ALTERAR OS TEXTOS E IMAGENS
------------------------------------------------------------
- Textos: Basta abrir os arquivos .html em um editor (VS Code, 
  Sublime ou Bloco de Notas) e alterar o conteúdo entre as tags.
- Ícones: Utilizamos a biblioteca FontAwesome. Você pode trocar 
  os ícones buscando por nomes no site fontawesome.com.
- Links: Procure por 'href="seu-link"' para direcionar os 
  botões para suas redes sociais ou checkout real.

------------------------------------------------------------
⚡ PERFORMANCE E SEO
------------------------------------------------------------
- Imagens: Recomendamos usar o formato .webp para carregamento 
  ultra-rápido.
- SEO: Altere as tags <title> e <meta name="description"> no 
  topo de cada arquivo HTML para melhorar seu ranking no Google.
